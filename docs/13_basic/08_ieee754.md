## 개요

컴퓨터에서의 실수 표현은 복잡하고, 그 개념이 방대하다.\
실수 표현의 연산은 근사값으로 처리하기 때문에\
정밀한 계산이 필요한 부분에서는 가능하면 실수 표현의 연산은 피하는 것이 좋겠지만,\
그럴 수 없는 경우가 많으니 컴퓨터에서의 실수 표현과 처리에 대해서 알아보자.

대부분의 실수 시스템은 `IEEE754` 표준을 따른다.\
1985년도에 최초로 개정이 되었으며, 2008년에 `IEEE754-2008` 버전이 개정되고,
최신은 2019년도 7월에 개정된 `IEEE754-0219` 버전이 있다.

다음 표준 버전은 2028년에 진행한다고 한다.

## 형식

IEEE754 형식은 `수치적인 값과 기호를 표현하는 집합`이라고 볼 수 있다.

32bit 체계와 64bit 체계로 나뉘며, 각 차이점은 아래와 같다.

| 기능   | 32bit | 64bit |
| ------ | ----- | ----- |
| min    |       |       |
| max    |       |       |
| 지수부 |       |       |
| 가수부 |       |       |

## 구조

부동 소수점 표현으로 3가지로 구성되어 나타낸다.

1. 부호 비트 (signed bit)
2. 지수 (exponent)
3. 가수 (fraction/mantissa)

-118.625 십진수를 나타내보자.

### 부호 비트

음수의 경우 부호 비트는 1로 설정된다.\
데이터 크기에 상관 없이 항상 최상위 1비트를 사용한다.\
`-118.625`는 음수이므로 `1`이 된다.

### 지수

절대 값인 `118.625`을 이진수로 나타내면 `1110110.101`이 된다.\
소수점을 왼쪽으로 이동시키고 지수부로 표현하면 아래와 같이 표현된다.\

`1.110110101` * `2^6`
